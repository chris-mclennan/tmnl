Or paste the shell installer for your platform:

```sh
# macOS / Linux
curl --proto '=https' --tlsv1.2 -LsSf https://github.com/chris-mclennan/tmnl/releases/latest/download/tmnl-rs-installer.sh | sh
```

```powershell
# Windows
irm https://github.com/chris-mclennan/tmnl/releases/latest/download/tmnl-rs-installer.ps1 | iex
```

## Platform details

<CardGrid>
  <Card title="macOS" icon="apple">
    **DMG** — drag `tmnl.app` to Applications, then click the dock icon for the GPU-rendered terminal.

    Or **`.pkg`** for the CLI install (drops `tmnl` on `PATH`). Per-arch — pick Apple Silicon or Intel from the [latest release](https://github.com/chris-mclennan/tmnl/releases/latest).

    Requires macOS 10.14 (Mojave) or newer.
  </Card>
  <Card title="Linux" icon="linux">
    Shell installer drops `tmnl` on `PATH`:

    ```sh
    curl -LsSf https://github.com/chris-mclennan/tmnl/releases/latest/download/tmnl-rs-installer.sh | sh
    ```

    You'll need a working OpenGL / Vulkan stack for `wgpu` to render. Most modern distros have this; check with `glxinfo` or `vulkaninfo` if tmnl fails to launch.
  </Card>
  <Card title="Windows" icon="seti:windows">
    Double-click the `.msi` from the [latest release](https://github.com/chris-mclennan/tmnl/releases/latest), or PowerShell:

    ```powershell
    irm https://github.com/chris-mclennan/tmnl/releases/latest/download/tmnl-rs-installer.ps1 | iex
    ```
  </Card>
  <Card title="Family bundle" icon="puzzle">
    tmnl pairs with [mnml](https://mnml.sh) (TUI IDE) and [mixr](https://mixr.sh) (DJ app). With all three installed, clicking the mnml or mixr dock icon opens them as native tabs inside tmnl.
  </Card>
</CardGrid>

## Build from source

```sh
git clone https://github.com/chris-mclennan/tmnl
cd tmnl
cargo build --release
./target/release/tmnl
```

Build deps:

- **macOS**: Xcode command-line tools (`xcode-select --install`)
- **Linux**: X11 + Wayland + GTK headers — `sudo apt install libx11-dev libxcursor-dev libxrandr-dev libxi-dev libxkbcommon-dev libgl1-mesa-dev libwayland-dev libudev-dev libfontconfig1-dev libglib2.0-dev libgtk-3-dev libxdo-dev`
- **Windows**: MSVC build tools (included with Visual Studio 2022 or the standalone Build Tools installer)

## Verify

```sh
tmnl --version
```

If your shell can't find `tmnl`, you may need to restart it (the installer puts `tmnl` on `PATH` via your shell profile).
